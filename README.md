#  Flight Automation Framework (Selenium + Java + Maven)

An end-to-end UI automation project built using **Selenium WebDriver**, **TestNG**, and **Maven**.  
It automates flight search and booking flows on Cleartrip.

##  Tech Stack
- Java 21
- Selenium 4
- Maven 3.9+
- TestNG
- WebDriverManager

##  Setup
```bash
mvn clean test
